Welcome to GitHub Pages.

大家是否为markdown没办法生成toc（Table of Content）而苦恼？

i5ting_ztree_toc是一个解决上面问题的插件，可以根据h1到h6自动生成toc，并且可以自动编号 


			### 安装

			```
				gem install tocmd
			```

			### 用法

			指定单个文件

			```
				tocmd -f shiti.md
			```
	
			指定目录

			```
				tocmd -d .
			```

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.

Usage

Add this line to your html file:

<script src="js/jquery-1.4.4.min.js"></script>
<script src="js/jquery.ztree.core-3.5.js"></script>
<script src="ztree_toc.js"></script>

Add this line to your html file:

 <link rel="stylesheet" href="zTreeStyle.css" type="text/css">

And then execute:

<script type="text/javascript" >
<!--
$(document).ready(function(){
    $('#tree').ztree_toc({

    });
});
//-->
</script>

Example

$(document).ready(function(){
    $('#tree').ztree_toc({
        is_auto_number: true,
        // documment_selector: '.first_part'
    });
});
